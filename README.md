<h2>🧑‍💻 About Me</h2>

<p>System-oriented Software Developer / Full-stack Developer with <b>13+ years of experience</b> designing, building, and maintaining enterprise applications across document management, workflow automation, financial reporting, digital services, transportation, e-wallet, toll payment, and aviation fuel operation domains.</p>

<p>I approach software by understanding the underlying model first: data flow, business rules, system boundaries, failure cases, and long-term maintainability. My work spans backend APIs, frontend applications, mobile WebView experiences, integration services, real-time dashboards, reporting systems, authentication/authorization, database design, CI/CD, and production troubleshooting.</p>

<p>Technically strongest in <b>.NET/C#</b>, <b>Angular/TypeScript</b>, <b>ASP.NET Core</b>, relational databases, Docker-based environments, and enterprise integration.</p>
